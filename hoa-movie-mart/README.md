## âš™ï¸ Theme Options (Movie Mart Settings)

The theme includes a powerful settings panel under **Appearance > Movie Mart Settings**:

### 1. Branding & Identity
- **Logo Upload:** Custom logo for header and footer.
- **Favicon:** Custom site icon.

### 2. Header Style & Typography
- **Background Color:** Customize header background.
- **Padding:** Adjust header height.
- **Font Size:** Set custom typography sizing.
- **Colors:** Independent controls for link and hover colors.
- **Sticky Header:** Toggle fixed-position header.

### 3. Footer Settings
- **Footer Description:** Custom "About Us" text for the footer.
- **Column Management:** Customize titles and extra text for Explore, Genres, and Legal columns.
- **Copyright Text:** Custom footer credit line.

### 4. API Configuration (Automation)
- **TMDB API Key:** Connects to The Movie Database for high-quality posters and metadata.
- **OMDB API Key:** Fetches precise IMDb ratings and runtimes.
- **TVmaze Integration:** Built-in support for TV series schedules.

### 5. Social Media & Advanced
- **Social Links:** Telegram and Facebook page integration.
- **Header Scripts:** Easy injection for Analytics, Pixels, or Ad scripts.
- **Cloudflare Turnstile:** Anti-bot protection for download links.

## ðŸ› ï¸ Installation & Setup

### Requirements
- WordPress 5.8 or higher.
- PHP 7.4 or higher.
- **HOA Movie Mart Core Plugin** (Required for CPTs and metadata).

### Quick Start
1. **Upload:** Upload the `hoa-movie-mart` folder to `/wp-content/themes/`.
2. **Activate:** Go to Appearance > Themes and activate **HOA Movie Mart**.
3. **Menus:**
   - Create a menu and assign it to the **Primary Menu** location.
   - Assign a menu to the **Mobile Menu** location for the side drawer.
4. **Pages:** Create a "Request" page using the **Request Page** template.
